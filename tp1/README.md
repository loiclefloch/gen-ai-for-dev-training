# TP — Spec Driven Development - Site de e-commerce

L’exercice consiste à implémenter les pages principales d’un mini-site e-commerce à partir de spécifications fonctionnelles.

L'objectif est de suivre la méthodologie présentée pour générer le code correspondant grâce à l’IA, le vérifier, le compléter.

---

## Spécifications fonctionnelles du site

- **spec_home_page.md**  
  Spécification fonctionnelle de la home : mise en avant, recherche intégrée, gestion des résultats.

- **spec_recherche_produits.md**  
  Spécification fonctionnelle de la page de recherche : filtres, tri, cas nominaux et cas limites.

- **spec_consultation_produit.md**  
  Spécification fonctionnelle de la fiche produit : images, description, prix, disponibilité.
