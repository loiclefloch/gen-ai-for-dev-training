# Formation Gen AI pour les développeurs

Un dépôt de formation contenant des travaux pratiques (TPs) pour apprendre et pratiquer les concepts et outils d'IA générative pour les développeurs.

## Structure

Ce dépôt est organisé en trois travaux pratiques :

- **tp1/** - Travail Pratique 1 - Développement piloté par les spécifications (Spec Driven Development)
- **tp2/** - Travail Pratique 2 - Rétro-ingénierie
- **tp3/** - Travail Pratique 3 - Refactoring

Chaque dossier de TP contient son propre README avec des instructions et des objectifs spécifiques.

## Pour commencer

Naviguez vers chaque dossier de TP pour accéder au matériel de l'exercice :

```bash
cd tp1  # Pour le premier travail pratique
cd tp2  # Pour le second travail pratique
cd tp3  # Pour le troisième travail pratique
```

## Prérequis

Les prérequis seront spécifiés dans le fichier README de chaque TP individuel.
