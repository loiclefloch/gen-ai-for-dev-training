# Gen AI for Dev Training

A training repository containing practical exercises (TPs) for learning and practicing generative AI concepts and tools for developers.

## Structure

This repository is organized into three practical exercises:

- **tp1/** - Practical Exercise 1
- **tp2/** - Practical Exercise 2
- **tp3/** - Practical Exercise 3

Each TP folder contains its own README with specific instructions and objectives.

## Getting Started

Navigate to each TP folder to access the exercise materials:

```bash
cd tp1  # For the first practical exercise
cd tp2  # For the second practical exercise
cd tp3  # For the third practical exercise
```

## Prerequisites

Prerequisites will be specified in each individual TP's README file.
